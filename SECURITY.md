# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Gale, please report it privately by contacting us on [Discord](https://discord.gg/vkUqjJ2xMg).

Please do not report security vulnerabilities through public GitHub issues, pull requests, or discussions.

## Response

We will acknowledge receipt of your report within 48 hours and provide an estimated timeline for a fix.

## Disclosure

We ask that you do not publicly disclose the vulnerability until we have had a chance to address it.
